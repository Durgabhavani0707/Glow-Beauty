💄 **Glow Beauty Website**

A modern full-stack beauty services website built using **Flask, HTML, CSS, and JavaScript**.

---

 🌐 **Live Demo**

👉 https://glow-beauty.onrender.com/

 ✨ **Features**

* 💄 Beauty Services (Makeup, Mehendi, Hair Styling)
* 🛍️ Product Listing (Cosmetics & Accessories)
* 🛒 Add to Cart functionality
* 📅 Booking System (Appointment form)
* 🎨 Attractive UI with animations
* 📱 Responsive design


🛠️ **Tech Stack**

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* Deployment: Render


 📂 **Project** **Structure**


beauty_website/
│── app.py

│   ├── services.html

 🚀 **How to Run Locally**


git clone https://github.com/your-username/Glow-Beauty.git
cd Glow-Beauty
pip install -r requirements.txt
python app.py
